# Project Brief: HostingTR

## Proje Adı
HostingTR - Hosting & Domain Hizmetleri Web Sitesi

## Proje Özeti
Türkiye pazarına yönelik, modern ve profesyonel bir hosting/domain şirketi web sitesi. Site, domain kaydı, web hosting, sunucu kiralama ve ilgili hizmetleri sunacak.

## Hedef Kitle
- Bireysel web sitesi sahipleri
- KOBİ'ler
- Kurumsal şirketler
- Web geliştiriciler ve ajanslar
- Reseller'lar

## Ana Özellikler
1. Domain arama ve kayıt sistemi
2. Hosting paket seçimi ve karşılaştırma
3. Sunucu yapılandırma
4. Online sepet ve ödeme
5. Müşteri girişi/kayıt
6. Blog ve destek içerikleri

## Başarı Kriterleri
- Hızlı yüklenen sayfalar (<2s)
- Mobile-first responsive tasarım
- Yüksek dönüşüm oranı (CTA'lar)
- SEO uyumlu yapı
- Erişilebilirlik standartları

## Kapsam Dışı
- Backend API geliştirme
- Ödeme gateway entegrasyonu
- WHMCS entegrasyonu
- Canlı destek sistemi

## Zaman Çizelgesi
Belirlenmedi - Faz bazlı ilerleme
